##### 배포 파일 실행 명령어

```
echo start
set BUILD_ID=dontKillMe
cd C:\Users\JH\Desktop\dev\spring\approval\build\libs
start /b java -jar .\approval-0.0.1-SNAPSHOT.jar
```

- `dontKillMe` : 젠킨스가 실행한 하위 프로세스 종료 방지
